Configuring and building QDP++

We need to build QDP++ first. Let us make a build direcotry and go into it:

mkdir /home/bj/build/qdp; cd /home/bj/build/qdp

Now we need to configure QDP++. We want to install it into /home/bj/install/qdp_scalar_single (scalar build in single precision). We will do it with the following command: 

/home/bj/src/qdp++/configure --prefix=/home/bj/install/qdp_scalar_single \
    --enable-parallel-arch=scalar

This should configure QDP++. By default the precision will be single precision. There is no optimisation, no extra flags. You should see lots of messages from configure telling you that it is looking for libxml etc. If configure exits without an error you should be able to type 

gmake; gmake install

To build the system.
